Preparing the camera

Insert SD Memory Card.


- SD카드는 포함되어 있지 않으니, 시판되고 있는 SD카드를 구입합니다(32GB까지 지원)


Connect the AC adaptor to the camera.

Date and Time setting

카메라를 켜고 SET 버튼을 누르면 시계 설정 화면이 나타납니다.
기록 또는 표시 방법 : 재생 화면에서 MENU 버튼을 두 번 누릅니다.


Set the date

1. "DATE"를 선택합니다.
2. SET 버튼을 누릅니다.(날짜 설정 화면이 나타납니다.)
3. 날짜를 임의로 설정합니다.(순서대로 연도, 월, 일)
4. SET 버튼을 누르면 저장됩니다.


Set the clock

1. "TIME"을 선택합니다.
2. SET 버튼을 누릅니다.(시간 설정 화면이 나타납니다.)
3. 시간을 임의로 설정합니다.(시간⟶ 분 순서로 설정, 24시간 시계는 시간 표시에 사용됩니다.)
4. SET 버튼을 누르면 저장됩니다.


Set the order in which the date is displayed during playback

1. "DISP"를 선택합니다.
2. SET 버튼을 누릅니다.(날짜 형식 설정화면이 나타납니다)
3. SET버튼을 위로 전환하여 Y/M/D ⟶ M/D/Y ⟶ D/M/Y(아래 버튼은 역순입니다.)
4. SET 버튼을 누르면 저장됩니다.

SWITCHING THE OPERATING MODE_NORMAL Mode Recording Menu1

촬영 메뉴 1에서는 동영상 해상도, 사진 해상도, 장면 모드, 필터 효과, 플래시 기능, 셀프 타이머
등을 설정할 수 있습니다. 동영상은 1,280 x 720 HD 해상도 영상을 초당 30프레임으로 찍거나
640 x 480 VGA 해상도 영상을 초당 60프레임으로 찍을 수 있습니다.
사진 해상도는 TH1의 기본 화소수인 110만 화소 외에도 화소 보간을 통한 200만 화소
해상도를 설정할 수 있습니다. 하지만 결과물이 110만 화소 사진보다 못한 경우가 많습니다.

&:No filter
%:Cosmetic filter
W:Monochrome filter
❀:Sepia filter

SWITCHING THE OPERATING MODE_NORMAL Mode Recording Menu2

촬영 메뉴 2에서는 손떨림 보정, 포커스 방식, AF 포커스 모드, 측광 방식,
ISO 감도 등을 설정할 수 있습니다. 포커스 방식은 표준, 매크로,
수동 초점을 선택할 수 있고, 포커스 모드는 9점 AF와 중앙 스팟 AF를 지원합니다.
감도는 ISO 50 부터 ISO 1600까지 설정할 수 있습니다.

SWITCHING THE OPERATING MODE_NORMAL Mode Recording Menu3

촬영 메뉴 3에서는 화이트 밸런스, 노출 모드, 얼굴 인식 기능,
고감도 지원 여부, 디지털 줌 이용 여부를 설정할 수 있습니다
